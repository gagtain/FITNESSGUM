# ФИТНЕСС ЗАЛ
# Цель создания:
Целью создания данного проекта явзяется привлечение интереса молодежи к спортивному образу жизни
# Описание основных разделов
Главная:  
Эта страница приветствует пользователя и имеет кнопку около центра экрана по нажатию на которую пользователь переходит на страницу Тренировки  
Тренировки:  
Эта страница в основном состоит из обьектов тренировок, при нажатии на которые можно перейти на страницу тренировки  
Тренировка:  
Эта страница состоит из описания дня тренировки и самих упражнений  
О нас:  
Эта страница содержит информацию о проекте  
Тест:  
На этой странице пользователь может пройти тест и получить рекомендацию по выбору тренироки, нажимая на рекомендацию пользователь переходит на страницу тренировки  
# Основная аудитория сайта:
Основной аудиторией сайта являются люди средней и старшей возрастной категории
