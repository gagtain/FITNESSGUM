# ФИТНЕСС ЗАЛ
с указанием наименования, цели создания, описания целевой аудитории сайта, описания содержания основных разделов, а также структуры сайта в виде mindmap.
# Цель создания:
Целью создания данного проекта явзяется привлечение интереса молодежи к спортивному образу жизни
# Описание основных разделов
Главная:  
Эта страница приветствует пользователя и имеет кнопку около центра экрана и по нажатию на неё пользователь переходит на страницу Тренировки
Тренировки:  
Эта страница в основном состоит из обьектов тренировок, при нажатии на которые можно перейти на страницу тренировки
Тренировка:  
Эта страница состоит из описания дня тренировки и самих тренировок
О нас:  
Эта страница имеет информацию о проекте
Тест:  
На этой странице пользователь может пройти тест и получить рекомендацию по выбору тренироки, нажимая на рекомендацию пользователь переходит на страницу тренировки
